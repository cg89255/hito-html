XML


Historia

XML fue desarrollado a finales de la década de 1990 por un grupo de expertos del World Wide Web Consortium (W3C). Se creó como un lenguaje de marcado diseñado para estructurar, almacenar y transportar datos de manera legible tanto por humanos como por máquinas. XML surgió como una alternativa a los formatos de datos más rígidos y específicos, como el HTML, que estaban diseñados principalmente para la presentación de información en la web.


Normas de uso

  • Jerarquía de datos: Los datos se organizan en una estructura jerárquica de etiquetas anidadas, similar a un árbol
  • Marcar los datos: Se utilizan etiquetas para definir la estructura y el significado de los datos.
  • Lenguaje legible por máquinas y humanos: XML está diseñado para ser legible tanto por humanos como por máquinas, lo que facilita su comprensión y procesamiento por parte de sistemas informáticos.
  • Extensible: Permite la definición de etiquetas personalizadas y estructuras de datos específicas para un dominio particular.
  • Independiente del lenguaje y la plataforma: XML puede ser utilizado en una amplia variedad de lenguajes de programación y sistemas operativos.

Sintáxis

La sintaxis básica de XML se compone de elementos y atributos. Un elemento tiene una etiqueta de apertura y una de cierre, y puede contener texto y otros elementos anidados. Los atributos proporcionan información adicional sobre un elemento y se especifican dentro de la etiqueta de apertura del elemento


Aplicación empresarial

  • Integración de sistemas: XML se utiliza para definir formatos de datos estándar que facilitan la comunicación entre sistemas empresariales que utilizan diferentes tecnologías y plataformas
  • Interoperabilidad: Al adoptar estándares basados en XML, las empresas pueden garantizar la interoperabilidad entre sus sistemas y los de sus socios comerciales.
  • Representación de datos estructurados: XML se utiliza para representar datos estructurados de manera legible y accesible, lo que facilita su procesamiento y análisis por parte de sistemas informáticos.
  • Web services: XML es fundamental en la implementación de servicios web, donde se utiliza para describir los mensajes intercambiados entre clientes y servidores, así como las interfaces de los servicios.
